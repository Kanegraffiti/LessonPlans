CONTACT 1
Date: __________________
Class: JSS2 (Basic 8)
Duration: 40 minutes
Period: 1st Period
Subject: Computer Studies
Topic: Programming Language
Subtopic: Meaning of computer program and programming language
Reference Materials:
- NERDC Computer Studies Curriculum for Upper Basic Education (JSS2)
- State Scheme of Work for Second Term Computer Studies
- Approved Computer Studies textbooks for Junior Secondary Schools
Instructional Materials:
- Flash cards of programming terms
- Chart showing categories of programming languages
- Whiteboard and marker
- Computer system (for demonstration where available)
Previous Knowledge:
- Learners have basic understanding of computer operations.
Entry Behaviour:
- Learners can state that computers follow instructions.
Behavioural Objectives:
By the end of the lesson, learners should be able to:
- define computer program.
- define programming language.
- identify who a programmer is.
- list the three main categories of programming languages.
- distinguish machine language from high-level language.
Lesson Presentations:
Step 1: Introduction/Set Induction
- Teacher asks learners how ATM or calculator follows commands.
- Teacher links responses to computer programs and programming language.
Step 2: Presentation/Teaching Steps
- Teacher explains the meaning of computer program and programming language.
- Teacher defines programmer and programming.
- Teacher introduces machine language, low-level language, and high-level language.
- Teacher compares features and uses of each category.
Step 3: Learners’ Activities
- Learners define key terms orally and in writing.
- Learners complete a comparison table in pairs.
- Learners provide examples of high-level languages they know.
Step 4: Summary/Board Summary
- Meaning of computer program.
- Meaning of programming language.
- Categories of programming languages.
- Basic differences among categories.
Lesson Content:
Computer Program
- A computer program is a set of instructions written to make a computer perform specific tasks.

Programming Language
- A programming language is a formal language used to write computer instructions.

Programmer and Programming
- Programmer: person who writes computer programs.
- Programming: process of writing and arranging computer instructions.

Main Categories of Programming Languages
- Machine language.
- Low-level language.
- High-level language.
Evaluation:
1. Define computer program.
2. What is programming language?
3. Who is a programmer?
4. List three categories of programming languages.
5. State two differences between machine language and high-level language.
6. Why is high-level language easier for humans to use?
Conclusion:
- Teacher concludes that programming languages are tools for communicating instructions to computers.
Assignment:
1. Write and explain the meaning of computer program and programming language.
2. Compare machine language and high-level language in three points.
3. List five examples of high-level programming languages.

CONTACT 2
Date: __________________
Class: JSS2 (Basic 8)
Duration: 40 minutes
Period: 2nd Period
Subject: Computer Studies
Topic: Programming Language
Subtopic: Types of high-level programming languages and examples
Reference Materials:
- NERDC Computer Studies Curriculum for Upper Basic Education (JSS2)
- State Scheme of Work for Second Term Computer Studies
- Approved Computer Studies textbooks for Junior Secondary Schools
Instructional Materials:
- Classification chart of high-level languages
- Flash cards with language names (BASIC, FORTRAN, COBOL, SQL, Java)
- Whiteboard and marker
Previous Knowledge:
- Learners can define computer program and programming language.
Entry Behaviour:
- Learners can mention at least one high-level language.
Behavioural Objectives:
By the end of the lesson, learners should be able to:
- list at least five types of high-level languages.
- state one example under each type.
- describe uses of scientific and commercial languages.
- identify special-purpose language from given examples.
- classify selected languages into correct groups.
Lesson Presentations:
Step 1: Introduction/Set Induction
- Teacher recaps Contact 1 using quick oral quiz.
- Teacher displays language names and asks learners to group them.
Step 2: Presentation/Teaching Steps
- Teacher explains types of high-level languages: scientific, multi-purpose, commercial, command language, and special-purpose.
- Teacher gives examples and practical applications for each type.
- Teacher clarifies common misconceptions in classification.
Step 3: Learners’ Activities
- Learners classify language cards in groups.
- Learners present one use-case for each language type.
- Learners answer short application questions.
Step 4: Summary/Board Summary
- Types of high-level languages.
- Examples and applications.
- Classification practice.
Lesson Content:
Types of High-level Languages
- Scientific language: used for scientific/engineering calculations. Example: FORTRAN, BASIC.
- Multi-purpose language: used for many applications. Example: C++, Java, Visual Basic.
- Commercial language: used for business data processing. Example: COBOL.
- Command language for operating system: controls operating system tasks. Example: DOS commands.
- Special-purpose language: designed for specific tasks. Example: SQL.

Applications
- Science and engineering calculations.
- Business records and payroll.
- Database queries and report generation.
- System control and automation.
Evaluation:
1. List five types of high-level languages.
2. Give one example of scientific language.
3. State one use of COBOL.
4. What type of language is SQL?
5. Mention two examples of multi-purpose languages.
6. Explain one difference between commercial and scientific languages.
7. Classify BASIC, SQL, COBOL, and Java into their groups.
Conclusion:
- Teacher concludes that high-level languages are grouped by purpose and application area.
Assignment:
1. Write and explain the five types of high-level programming languages.
2. Draw and label a classification chart of programming languages.
3. Provide two real-life uses each for scientific and commercial languages.
4. Explain why programmers choose different languages for different tasks.
