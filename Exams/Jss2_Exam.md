# Christ's Chapel Foundation School Midterm Exams
## Combined Exam: Basic Science, Basic Technology, and Information Technology
### Class: JSS2
Name: ____________________    Date: ____________________

**Objective Section (Answer all questions).**

1. A habitat is best described as the place where an organism _____.
   a) hides from predators only  
   b) finds food, water, shelter, and mates  
   c) stores its eggs alone  
   d) travels for migration only
2. An organism adapted with gills is likely to live in a ____ habitat.
   a) Desert  
   b) Forest  
   c) Freshwater  
   d) Mountain peak
3. Which feature helps a cactus survive in the desert?
   a) Wide leaves for trapping water  
   b) Spines that reduce water loss  
   c) Smooth skin for sliding  
   d) Thick fur for warmth
4. The immediate care given to an injured person before medical help is _____.
   a) clinic service  
   b) first aid  
   c) rehabilitation  
   d) surgery
5. Which item is a must-have in a standard first aid box?
   a) Chalk  
   b) Scissors  
   c) Game cards  
   d) Newspaper
6. Cotton wool in first aid is mainly used to _____.
   a) write prescriptions  
   b) clean and pad wounds  
   c) close a bottle  
   d) decorate the box
7. Software in a computer refers to the _____.
   a) electrical cables  
   b) programs and instructions  
   c) metal casing  
   d) keyboard keys
8. Which of the following is application software?
   a) Operating system  
   b) Word processor  
   c) Power supply unit  
   d) Monitor
9. The part of the computer that displays output is the _____.
   a) Keyboard  
   b) Mouse  
   c) Monitor  
   d) Joystick
10. A balanced environment maintains stability because _____.
    a) only one species survives  
    b) population and resources are in harmony  
    c) predators disappear  
    d) humans remove all plants
11. Which human activity best protects habitats?
    a) Bush burning  
    b) Afforestation  
    c) Dumping refuse in rivers  
    d) Overfishing
12. An advantage of an ecosystem survey in school is to _____.
    a) reduce learning  
    b) identify local plants and animals  
    c) increase pollution  
    d) disturb wildlife only
13. A basic maintenance task for computer systems is to _____.
    a) delete antivirus software  
    b) clean dust from vents and keyboards  
    c) pour water on components  
    d) remove all cables while running
14. Which storage device is most suitable for carrying project files between school and home?
    a) DVD drive  
    b) Solid-state flash drive  
    c) Power cable  
    d) Printer
15. An example of a safety precaution in the workshop is to _____.
    a) wear loose clothing near machines  
    b) keep tools arranged and floors dry  
    c) run in the workshop  
    d) store nails on the floor

**Theory Section (Answer 3 questions: choose 1 from Basic Science, 1 from Basic Technology, and 1 from Information Technology).**

1. (Basic Science) Explain how three adaptations help animals survive in their habitats, using specific examples.
2. (Basic Science) Describe two ways human activities can upset the balance of an ecosystem and how these effects can be controlled.
3. (Basic Technology) Outline the steps for treating a minor cut in the workshop and list four items you would need.
4. (Basic Technology) Identify three workshop safety rules and explain why each rule is important.
5. (Information Technology) Describe the roles of system software and application software, giving one example of each.
6. (Information Technology) Explain how to care for computer hardware to keep it functioning properly and safe for users.
