CONTACT 1
Date: __________________
Class: JSS1 (Basic 7)
Duration: 40 minutes
Period: 1st Period
Subject: Basic Technology
Topic: Revision of Last Term’s Work and Board Practice
Subtopic: Drawing instruments and setting drawing paper on board
Reference Materials:
- NERDC Basic Technology Curriculum for Upper Basic Education (JSS1)
- State Scheme of Work for Second Term Basic Technology
- Approved Basic Technology textbooks for Junior Secondary Schools
Instructional Materials:
- Drawing board, drawing sheet, masking tape
- T-square, set squares, pencils, eraser, sharpener
- Samples of title block formats
- Whiteboard and marker
Previous Knowledge:
- Learners have been taught introductory drawing and basic tool handling.
Entry Behaviour:
- Learners can identify common drawing instruments.
Behavioural Objectives:
By the end of the lesson, learners should be able to:
- identify at least five drawing instruments.
- state two uses of T-square and set squares.
- demonstrate correct setting of drawing paper on board.
- sharpen a pencil to suitable point for technical drawing.
- draw simple horizontal and vertical guide lines.
Lesson Presentations:
Step 1: Introduction/Set Induction
- Teacher asks learners to name tools used during first term drawing lessons.
- Teacher displays the instruments and links to proper board practice.
Step 2: Presentation/Teaching Steps
- Teacher reviews key first-term concepts briefly.
- Teacher demonstrates instrument identification and functions.
- Teacher demonstrates how to fix drawing paper on board correctly.
- Teacher demonstrates pencil sharpening and proper holding of T-square.
Step 3: Learners’ Activities
- Learners identify instruments in groups.
- Learners practise fixing drawing paper and checking alignment.
- Learners sharpen pencils and draw short horizontal and vertical lines.
Step 4: Summary/Board Summary
- Names and uses of drawing instruments.
- Steps for setting drawing sheet.
- Basic rules for neat board practice.
Lesson Content:
Revision Highlights
- Accuracy and neatness in technical drawing.
- Correct use and care of drawing instruments.

Basic Board Practice
- Required instruments: drawing board, T-square, set square, pencil, eraser.
- Steps for setting paper:
  - place paper in position.
  - align edge with board guide.
  - tape corners firmly.
- Pencil sharpening:
  - conical point for general lines.
  - knife edge for precision lettering.
Evaluation:
1. List five instruments used in board practice.
2. State two uses of T-square.
3. Why is correct paper setting important?
4. Mention two types of pencil points used in technical drawing.
5. Demonstrate one proper method of holding a T-square.
Conclusion:
- Teacher emphasizes that neat board practice is the foundation of good technical drawing.
Assignment:
1. Write and explain the steps for setting drawing paper on board.
2. Draw and label five drawing instruments.
3. Practise drawing ten horizontal and ten vertical lines in your exercise book.

CONTACT 2
Date: __________________
Class: JSS1 (Basic 7)
Duration: 40 minutes
Period: 2nd Period
Subject: Basic Technology
Topic: Board Practice
Subtopic: Border lines, title block, and freehand lettering
Reference Materials:
- NERDC Basic Technology Curriculum for Upper Basic Education (JSS1)
- State Scheme of Work for Second Term Basic Technology
- Approved Basic Technology textbooks for Junior Secondary Schools
Instructional Materials:
- Drawing board and sheets
- T-square, set squares, pencils, eraser
- Sample title block chart
- Whiteboard and marker
Previous Knowledge:
- Learners can set drawing paper on board and identify drawing tools.
Entry Behaviour:
- Learners can draw basic straight lines with guidance.
Behavioural Objectives:
By the end of the lesson, learners should be able to:
- draw border lines using T-square and set squares.
- position and draw a simple title block correctly.
- write legible block letters and numerals freehand.
- maintain neatness and proper spacing in drawing work.
- present a complete simple drawing layout.
Lesson Presentations:
Step 1: Introduction/Set Induction
- Teacher asks learners to recall steps for preparing drawing board.
- Teacher displays a complete drawing sheet with border and title block.
Step 2: Presentation/Teaching Steps
- Teacher demonstrates drawing border lines at correct margins.
- Teacher demonstrates title block position and essential details.
- Teacher teaches freehand block lettering techniques.
- Teacher guides learners on spacing and line quality.
Step 3: Learners’ Activities
- Learners draw border lines on prepared sheets.
- Learners create title block and fill sample entries.
- Learners practise alphabets and numerals in block style.
Step 4: Summary/Board Summary
- Rules for border lines.
- Components of title block.
- Features of legible freehand lettering.
Lesson Content:
Border Lines
- Border lines frame the drawing and improve neatness.
- Use T-square and set square for straight and accurate edges.

Title Block
- Usually placed at lower right corner.
- Contains: title of drawing, name, class, date, scale.

Freehand Lettering
- Use upright block letters.
- Maintain equal size, spacing, and line thickness.
- Keep writing neat and readable.
Evaluation:
1. State the purpose of border lines in technical drawing.
2. Where is a title block commonly placed?
3. List four details found in a title block.
4. Why is legible lettering important?
5. Draw a simple title block layout.
6. Write A-Z and 0-9 in clear block letters.
Conclusion:
- Teacher concludes that proper layout and lettering improve drawing quality and communication.
Assignment:
1. Write and explain the importance of title block in technical drawing.
2. Draw and label a complete drawing sheet showing border lines and title block.
3. Practise one page of block letters and numerals.
