# LessonPlans

A static, inspector-ready viewer for JSS 1-3 lesson plans. Use the included landing page to browse combined weeks 1-10 content per subject and export a clean PDF copy.

## Getting started

1. Open `index.html` in your browser to preview the site locally.
2. Select **Basic Science**, **Basic Technology**, or **Information Technology** to load the full weekly plan pulled from the Markdown source files.
3. Expand any week block to review details. Click **Download PDF** to export the displayed content in a single document.

## Deploying to GitHub Pages

1. Commit and push the repository to GitHub.
2. In your repository settings, enable **Pages** and choose the `main` branch with the `/ (root)` folder.
3. Share the generated GitHub Pages URL with inspectors. All assets are self-contained and will render directly from the repository.
